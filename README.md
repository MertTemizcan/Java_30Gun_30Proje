# 30 Projects in 30 Days

 About This Repository (EN)

This repository contains projects I developed as part of my "30 Projects in 30 Days" personal development series.

The main goals of this series were:
- To strengthen my programming fundamentals
- To improve my algorithmic thinking and problem-solving skills
- To gain daily coding discipline
- To transform theoretical knowledge into practical projects

Throughout the process, I developed small to medium-sized projects focusing on:
- Variables, conditions, and loops
- Receiving and validating user data
- Console-based and simple interface applications

Each project represents a specific learning outcome and demonstrates my progress over time.

The focus of this work was not on perfection, but on **continuity, learning, and project completion**.

Technologies Used
- Java
- Console Applications

This repository is a work where I declare my software development journey and which forms the basis for advanced projects.
